# 全球时间天气仪表盘

一个现代化的本地网页应用，显示多个城市的时间、天气信息，并提供强大的计时器功能。

## 功能特性

### 时间与天气显示
- 📍 显示本地城市（可自定义选择）、巴黎、纽约的实时时间
- 🌤️ 显示温度、湿度、天气状况图标、空气质量指数
- 🔄 自动更新时间（每秒）和天气（每10分钟）

### 计时器功能
- ⏱️ **正计时模式**：秒表功能，从0开始累加
- ⏲️ **倒计时模式**：定时器功能，设定时间倒数
- ⏸️ 支持开始、暂停、重置操作
- 🏁 **计圈功能**：记录阶段时间和累计时间
- 🔔 倒计时结束时播放提示音并屏幕闪烁

### 界面设计
- 🎨 现代简洁的设计，适合办公和会议场景
- 📱 完全响应式，支持桌面、平板、手机
- 🌈 专业配色，视觉舒适

## 快速开始

### 1. 获取 OpenWeatherMap API Key

1. 访问 [OpenWeatherMap](https://openweathermap.org/api)
2. 点击右上角 **Sign Up** 注册账号
3. 登录后，进入 [API Keys 页面](https://home.openweathermap.org/api_keys)
4. 复制默认的 API Key（或创建新的）

**免费版限制：**
- 每分钟60次调用
- 每天1,000,000次调用
- 对于本项目完全够用

### 2. 配置 API Key

1. 打开 `script.js` 文件
2. 找到第2行：
   ```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
   ```
3. 将 `YOUR_API_KEY_HERE` 替换为你的实际 API Key：
   ```javascript
   const API_KEY = 'abc123def456...'; // 你的API Key
   ```

### 3. 运行应用

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js (需要先安装 http-server)
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 使用指南

### 选择本地城市
1. 点击本地城市卡片上的 📍 按钮
2. 在弹出的对话框中搜索或选择城市
3. 选择后会自动更新天气信息

### 使用计时器

#### 正计时（秒表）
1. 点击"正计时"按钮
2. 点击"开始"按钮开始计时
3. 点击"计圈"记录阶段时间
4. 点击"暂停"暂停计时
5. 点击"重置"清空计时

#### 倒计时（定时器）
1. 点击"倒计时"按钮
2. 在输入框中设置时间（时:分:秒）
3. 点击"开始"开始倒计时
4. 时间到达后会播放提示音并闪烁屏幕
5. 可随时"暂停"或"重置"

### 计圈功能
- 在计时过程中点击"计圈"按钮
- 会记录：
  - **阶段时间**：本圈用时
  - **累计时间**：总用时
- 计圈记录会显示在下方列表中

## 注意事项

1. **API Key 安全**：本项目为纯前端应用，API Key 会暴露在代码中。OpenWeatherMap 免费版已足够使用，但建议不要将此代码公开发布到互联网。

2. **首次加载**：第一次打开页面时，天气数据可能需要几秒钟加载。

3. **网络连接**：需要联网才能获取天气数据，时间显示功能离线也可使用。

4. **时区准确性**：时间显示基于浏览器的时区数据库，通常非常准确。

## 自定义

### 添加更多城市
编辑 `index.html` 中的城市选项列表：
```html
<div class="city-option" data-city="城市名" data-lat="纬度" data-lon="经度">城市名</div>
```

### 修改配色
编辑 `styles.css` 中的 CSS 变量：
```css
:root {
    --primary-bg: #f5f7fa;
    --accent-blue: #3498db;
    /* 更多颜色变量... */
}
```

### 调整天气更新频率
编辑 `script.js` 中的更新间隔（单位：毫秒）：
```javascript
setInterval(updateAllWeather, 600000); // 600000ms = 10分钟
```

## 故障排除

### 天气数据显示"无法获取数据"
- 检查是否正确配置了 API Key
- 检查网络连接
- 打开浏览器开发者工具（F12）查看控制台错误信息
- 确认 API Key 是否有效（可能需要等待几分钟激活）

### 计时器不工作
- 刷新页面重试
- 检查浏览器控制台是否有 JavaScript 错误

### 提示音不播放
- 某些浏览器需要用户交互后才能播放音频
- 检查浏览器是否静音
