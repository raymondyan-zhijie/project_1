# 全球时间天气仪表盘

一个现代化的本地网页应用，专为**会议室展示**优化，显示多个城市的时间，并提供超大字体的计时器功能。支持完全离线运行。

## ✨ 核心特性

### 🎯 会议室展示优化
- **超大计时器显示**：12rem字体，远距离清晰可见

- **1080P优化**：专为1080P投影/显示器设计，一屏完整显示
- **智能主题**：自动根据日出日落时间切换白天/黑夜主题

### ⏱️ 强大的计时器功能
- **正计时模式**：秒表功能，从0开始累加
- **倒计时模式**：定时器功能，设定时间倒数
- **计圈功能**：记录阶段时间和累计时间
- **大字体显示**：12rem超大字体，会议室远处可见
- **大按钮操作**：1.8rem字体，180px最小宽度，易于操作
- **提示音功能**：倒计时结束时播放提示音并屏幕闪烁

- **离线运行**：支持最长7天完全离线使用

- **数据持久**：使用localStorage永久保存

详细说明请查看 [离线缓存系统文档](OFFLINE_CACHE_README.md)

### 🌓 智能日夜主题
- **精确判断**：基于真实日出日落时间（来自OpenWeatherMap API）
- **自动切换**：白天显示浅色卡片，夜间显示深色卡片
- **离线支持**：使用缓存数据或时区时间判断
- **实时更新**：每次获取天气数据时同步更新

- 🇨🇳 中文
- 🇬🇧 English

## 🚀 快速开始

### 1. 获取 OpenWeatherMap API Key（可选，用于在线功能）

1. 访问 [OpenWeatherMap](https://openweathermap.org/api)
2. 点击右上角 **Sign Up** 注册账号
3. 登录后，进入 [API Keys 页面](https://home.openweathermap.org/api_keys)
4. 复制默认的 API Key（或创建新的）

**免费版限制：**
- 每分钟60次调用
- 每天1,000,000次调用
- 对于本项目完全够用

**注意**：如果不配置API Key，应用仍可完全离线运行，但会使用简化的时间判断（6:00-18:00为白天）。

### 2. 配置 API Key（可选）

1. 打开 `config.js` 文件
2. 找到第3行：
   ```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
   ```
3. 将 `YOUR_API_KEY_HERE` 替换为你的实际 API Key：
   ```javascript
   const API_KEY = 'abc123def456...'; // 你的API Key
   ```

### 3. 运行应用

#### 方法1：直接打开（推荐）
双击 `index.html` 文件，在浏览器中打开即可。

#### 方法2：使用本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server
```
然后在浏览器中访问 `http://localhost:8000`

### 4. 会议室使用建议

1. **全屏显示**：按 `F11` 进入全屏模式
2. **调整缩放**：使用 `Ctrl + +/-` 调整浏览器缩放，获得最佳显示效果
3. **首次联网**：首次使用时建议联网，让系统缓存所有城市的日出日落数据
4. **定期更新**：建议每周联网一次，更新缓存数据

## 📖 使用指南

### 选择本地城市
1. 点击第一个城市卡片上的 📍 按钮
2. 在弹出的对话框中搜索或选择城市
3. 选择后会自动更新该卡片的时间显示

### 使用计时器

#### 正计时（秒表）
1. 点击"正计时"按钮
2. 点击"开始"按钮开始计时
3. 点击"计圈"记录阶段时间
4. 点击"暂停"暂停计时
5. 点击"重置"清空计时

#### 倒计时（定时器）
1. 点击"倒计时"按钮
2. 在输入框中设置时间（时:分:秒）
3. 点击"开始"开始倒计时
4. 时间到达后会播放提示音并闪烁屏幕
5. 可随时"暂停"或"重置"

### 计圈功能
- 在计时过程中点击"计圈"按钮
- 会记录：
  - **阶段时间**：本圈用时
  - **累计时间**：总用时
- 计圈记录会显示在下方列表中

### 离线使用
1. **首次联网**：首次访问时联网，系统自动缓存所有城市的日出日落数据
2. **断网使用**：之后可以完全离线使用，主题切换仍然准确
3. **定期联网**：建议每周联网一次更新缓存（非必需）

## ⚙️ 自定义配置

### 添加更多城市

在 `script.js` 的 `cities` 对象中添加：

```javascript
newcity: {
    name: 'City Name',      // 英文名
    nameCN: '中文名',        // 中文名
    lat: 纬度,              // 纬度
    lon: 经度,              // 经度
    timezone: 'Area/City'   // IANA时区名称
}
```

系统会自动：
- 获取该城市的日出日落数据
- 保存到本地缓存
- 支持离线判断

### 修改配色

编辑 `styles.css` 中的 CSS 变量：
```css
:root {
    --primary-bg: #f5f7fa;
    --accent-blue: #3498db;
    --card-bg-day: #ffffff;
    --card-bg-night: #1a2332;
    /* 更多颜色变量... */
}
```

### 调整计时器大小

在 `styles.css` 中修改：
```css
.timer-display {
    font-size: 12rem;  /* 调整计时器字体大小 */
}
```

### 调整天气更新频率

编辑 `script.js` 中的更新间隔（单位：毫秒）：
```javascript
setInterval(updateAllWeather, 600000); // 600000ms = 10分钟
```

## 🔍 故障排除

### 主题颜色不正确
- **在线模式**：检查API Key是否配置正确
- **离线模式**：首次使用时需要联网，缓存日出日落数据
- **调试**：打开控制台（F12）查看缓存状态
  ```javascript
  JSON.parse(localStorage.getItem('sunriseSunsetCache'))
  ```

### 天气数据显示"无法获取数据"
- 检查是否正确配置了 API Key
- 检查网络连接
- 打开浏览器开发者工具（F12）查看控制台错误信息
- 确认 API Key 是否有效（可能需要等待几分钟激活）
- **注意**：离线模式下天气数据不可用，但时间显示和主题判断仍正常

### 计时器不工作
- 刷新页面重试
- 检查浏览器控制台是否有 JavaScript 错误

### 提示音不播放
- 某些浏览器需要用户交互后才能播放音频
- 检查浏览器是否静音

### 清除缓存
如果遇到数据异常，可以清除缓存重新获取：
```javascript
// 在浏览器控制台执行
clearSunriseSunsetCache();
```
